## 部署步骤

### 1. 启动后端服务器
```bash
cd /root/.openclaw/workspace/stock-analysis-system/backend
python3 app.py
```

后端服务器将在 http://localhost:5000 运行

### 2. 前端依赖安装
```bash
cd /root/.openclaw/workspace/stock-analysis-system/frontend
npm install
```

### 3. 启动前端开发服务器
```bash
cd /root/.openclaw/workspace/stock-analysis-system/frontend
npm start
```

前端将在 http://localhost:3000 运行

### 4. API测试
```bash
curl http://localhost:5000/api/stock/daily
curl http://localhost:5000/api/stock/weekly
curl http://localhost:5000/api/fundflow/sector
curl http://localhost:5000/api/ai/predictions
curl http://localhost:5000/api/stock/sector-link
curl http://localhost:5000/api/strategy/recommend
```
