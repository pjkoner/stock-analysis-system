## GitHub仓库创建步骤

### 1. 登录GitHub
```bash
gh auth login
```

### 2. 创建仓库
```bash
gh repo create stock-analysis-system --public --description "股票分析系统"
```

### 3. 推送代码
```bash
git remote add origin https://github.com/<username>/stock-analysis-system.git
git push -u origin main
```

## 本地部署步骤

### 1. 启动后端
```bash
cd backend
python3 app.py
```

### 2. 启动前端
```bash
cd frontend
npm install
npm start
```
